# 🏠 Tongkonan Structure Interactive Website

Website edukasi interaktif yang menjelaskan struktur rumah adat Toraja **Tongkonan** secara visual dan mudah dipahami.

---

## 📌 Deskripsi

Project ini dibuat untuk membantu pengguna memahami 3 bagian utama Tongkonan, yaitu:

* **Rattiang Banua** (Bagian Atas / Atap)
* **Kale Banua** (Bagian Tengah / Badan Rumah)
* **Sulluk Banua** (Bagian Bawah / Kolong)

Website dilengkapi dengan tampilan visual seperti infografis serta fitur interaktif dan kuis.

---

## ✨ Fitur Utama

* 🎨 Desain modern dengan tema budaya (earth tone)
* 🏠 Visual struktur Tongkonan (interaktif)
* 🖱️ Hover / klik untuk melihat penjelasan tiap bagian
* 🧠 Kuis pilihan ganda (10 soal)
* 📊 Perhitungan skor otomatis
* 📘 Penjelasan jawaban benar & salah
* 📱 Responsive (bisa dibuka di HP & desktop)

---
